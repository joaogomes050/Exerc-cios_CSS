Seletores

Seletor de TAG

Ao usar a seletor tipo tag em H1 e H2 todos os elementos desse tipo vão herdar os estilos definidos pelo código.

Seletor tipo .id

O seletor #id deve ser usado uma única vez no arquivo ! Cada elemento pode ter apenas um ID. Cada página pode ter apenas um elemento com aquele ID.

Seletor de Classe

Qualquer informação de estilo que precisa ser aplicada a múltiplos elementos em uma página dever ser feita com uma classe. Você pode usar a mesma classe para vários elementos. Você pode usar várias classes para um mesmo elemento.
